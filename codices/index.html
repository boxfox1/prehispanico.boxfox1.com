Cómo leer esta sección

Los códices aquí no se entienden solo como piezas históricas, sino como sistemas complejos de representación visual y documental. Esta sección sirve para ubicar cada manuscrito dentro de su cultura, materialidad, función, cronología y lógica de lectura.

También funciona como puente entre culturas, deidades, sistema visual, glosario y casos aplicados.

Estructura base del archivo documental

Identificación
Título, cultura, cronología, ubicación actual y contexto general.
Materialidad
Soporte, formato, técnica, pigmentos y construcción física del manuscrito.
Función
Calendario, ritual, historia, genealogía, registro o conocimiento específico.
Sistema visual
Escritura, organización de imagen, lectura interna y relaciones formales.
